# Architecture

## Purpose
CompliBot MVP is an AI-assisted compliance and SOP analysis tool.

## Core Entry Points
- `app.py` — main app entry point
- `compli_pipeline.py` — compliance processing pipeline

## Data Areas
- `data/uploads/` — uploaded files
- `data/runtime/vector_store/chroma_store/` — runtime vector storage

## Notes
This project is being normalized to the enterprise workspace standard while preserving current behavior.
